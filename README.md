Heroku Buildpack for Node.js and gulp.js
========================================

Usage
-----

- Set your Heroku app's buildpack URL to `https://github.com/bobvoorneveld/heroku-buildpack-nodejs-gulp.git`
- Add a Gulp task called `build` that builds your app
